# AI Tools Setup Documentation

## Approach & Thought Process

Instead of relying on a fixed tutorial, I approached this task by exploring the tools step-by-step and solving issues as they arose.

While setting up Cursor and extensions, I encountered multiple points of confusion (such as Git not being detected and multiple similar extensions in the marketplace). Instead of skipping these, I tried to understand the differences and chose the most reliable options.

This helped me not only complete the setup but also understand how these tools actually work in a real environment.

---

## Overview

This repository documents the process of setting up AI-assisted development tools using Cursor IDE, Claude Code, and Codex. The goal was to install the required tools, explore their functionality, and understand how they can be used in a development workflow.

- Explored interface including file navigation, terminal, and AI chat panel

### 2. Claude Code (Anthropic)
- Installed "Claude Code for VS Code" extension
- Explored its integration within the editor
- Attempted to understand how it supports AI-assisted coding

### 3. Codex (OpenAI)
- Installed "Codex – OpenAI’s coding agent"
- Logged in and explored its capabilities

## Issues Faced & Solutions

### Issue 1: Git not detected in Cursor
- Cursor showed "Git not found" error
- **Solution:** Used GitHub Desktop for repository management instead of relying on Cursor Git integration

### Issue 2: Difficulty finding correct extensions
- Multiple similar extensions appeared in marketplace
- **Solution:** Identified and installed official extensions from Anthropic and OpenAI

### Issue 3: Confusion between Cursor and VS Code
- Initially worked in VS Code instead of Cursor
- **Solution:** Switched to Cursor IDE to align with assignment requirements

## Key Learnings

- Setting up tools requires careful selection of correct extensions
- Official sources (Anthropic, OpenAI) are important for reliability
- Cursor provides built-in AI capabilities in addition to extensions
- Problem-solving and adapting to issues is a key part of the process
